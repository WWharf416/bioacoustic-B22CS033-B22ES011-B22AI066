    📖 How It Works

    🔊

    Few-Shot Learning

    Given just a few annotated audio examples (the support set), the model learns to detect similar bioacoustic events in unseen recordings — no fine-tuning required.

    🧬

    BEATs Encoder

    Audio is encoded frame-by-frame using Microsoft's BEATs model, producing 768-dimensional embeddings at ~48 fps (20.83ms per frame).

    🎯

    Prototypical Scoring

    Positive and negative prototypes are averaged from support embeddings. Each query frame is scored as cos(frame, pos) − cos(frame, neg).

    📈

    Event Decoding

    Scores are median-filtered, thresholded (adaptively or manually), and consecutive frames above threshold are merged into event segments.
